# usb-tree.github.io

A landing page for [USB Tree](https://github.com/AOzmond/usb-tree), built with Next.js and React, deployed via GitHub Pages at **https://usb-tree.github.io**.  

Built using the [Solid-Next.js template](https://nextjstemplates.com/templates/solid).

---

## Tech Stack

- **Framework:** Next.js 16
- **Language:** TypeScript
- **UI:** React 19
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Linting:** ESLint (Next.js config)
- **Formatting:** Prettier

---

## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed

### Install dependencies
```bash
 bun install
 ```
### Run the development server
```bash
 bun dev
 ```


## Deployment

This project is deployed to **GitHub Pages** at:

**https://usb-tree.github.io**

The deployment flow will typically:

1. Build the app with `bun run build`.
2. Publish the generated static output to the `gh-pages` or `main` branch configured for GitHub Pages.

Adjust this section to match your actual CI / deployment setup.
